# 🪔 사주 작명 (Saju Naming)

생년월일시 사주(만세력)를 분석해, 이름을 **6가지 항목**으로 채점하고 육각형 그래프로 보여주는 작명 웹앱입니다.

## 채점 6항목 (총점 100)
| 항목 | 가중치 | 설명 |
|------|--------|------|
| 사주보완 | 25 | 자원오행이 용신·부족오행을 보충 (간이 억부용신) |
| 수리길흉 | 20 | 81수리 4격(인·지·외·총) |
| 발음오행 | 20 | 성→이름 초성오행 상생 흐름 |
| 자원조화 | 15 | 이름 두 글자 자원오행 상생/비화 |
| 음양조화 | 10 | 획수 홀짝 음양 배열 |
| 발음편의 | 10 | 부르기 좋은 정도 |

## 기술
- 정적 웹앱 (빌드 불필요)
- 만세력: [lunar-javascript](https://github.com/6tail/lunar-javascript) (지방시·진태양시 보정 포함)
- 그래프: Chart.js (radar)
- 데이터: Supabase (한자 사전) — 미연결 시 로컬 시드(`data/hanja.js`)로 폴백

## 로컬 실행
```bash
python3 -m http.server 8777
# http://localhost:8777
```

## 면책
작명 로직은 유파별 차이가 있으며, 본 결과는 통용 성명학(억부용신·발음/자원오행·81수리) 기반 참고용입니다.
